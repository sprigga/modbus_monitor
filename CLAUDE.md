# Instructions

## Chinese (Original)
1. 執行或測試時要使用uv指令切換python環境.
2. 不要移除掉原有程式碼，要被修改的程式碼先註釋並說明，以方便之後追蹤和恢復

## English (Translation)
1. Use the `uv` command to switch Python environments when executing or testing.
2. Don't remove existing code; comment and explain modified code for easier tracking and restoration later.
